# tech.md 記載ルール

使用技術・ライブラリ・外部API・実行環境を記載する。

## フォーマット

```
## 使用言語・フレームワーク
- 言語: Python 3.13 など
- フレームワーク: FastAPI など

## 外部API・サービス連携
- サービス名: 用途・エンドポイント

## 依存ライブラリ
- ライブラリ名 バージョン: 用途

## 環境情報
- OS: Windows 11 など
- 実行環境: Lambda / EC2 / ローカル など

## 技術的な制約・注意事項
- 使用禁止の技術・バージョン固定の理由など
```

## ルール
- バージョンは可能な限り明記する
- 「なぜそのライブラリを使っているか」が不明な場合は「用途: 不明（要確認）」と記載する
- 制約事項は必ず理由とともに記載する
